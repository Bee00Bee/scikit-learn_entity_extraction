# scikit-learn entity recognition

### Project Description
In this project, I show the useful ways of working on entity recognition in the text using Tensorflow. 

### Required packages:
* [Tensorflow](https://tfhub.dev/)
* [pandas](https://pandas.pydata.org/)
* [itertools](https://docs.python.org/3/library/itertools.html)
* [nltk](https://www.nltk.org/)
* [eli5](https://pypi.org/project/eli5/)
* [collections](https://docs.python.org/3/library/collections.html)
* [skelrean packages](https://scikit-learn.org/stable/index.html)
  * [classification report](https://scikit-learn.org/stable/modules/generated/sklearn.metrics.classification_report.html)
* [sklearn_crfsuite](https://sklearn-crfsuite.readthedocs.io/en/latest/)
